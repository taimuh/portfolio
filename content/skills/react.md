# React

React は、ユーザーインターフェース構築のための JavaScript ライブラリです。本業での開発において 3 年間使用しており、コンポーネントベースの開発に深い理解を持っています。

## 主な経験

- **企業向けアプリケーション開発**: 大規模なエンタープライズアプリケーションでの React 開発
- **パフォーマンス最適化**: メモ化、lazy loading、コード分割などの最適化手法
- **状態管理**: Context API、Redux、Zustand などの状態管理ライブラリの活用
- **テスト**: Jest、React Testing Library を使用したユニットテスト

## 得意分野

- Hooks API の活用
- カスタム Hooks の設計・実装
- TypeScript との組み合わせ
- アクセシビリティ対応
